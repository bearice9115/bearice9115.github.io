Nodi (노디)

Secure Google Drive Backup & Sync Service

Application Purpose / 앱 목적

[English]
Nodi is a utility application designed to help users securely backup and sync their personal notes and configuration files to their own Google Drive storage (using Google Drive App Data Folder - appDataFolder). The app does not transmit any user data to external servers; all backups remain strictly within the user's private Google account.

[한국어]
Nodi(노디)는 사용자의 개인 메모 및 설정 데이터를 사용자의 개인 구글 드라이브 보관함(App Data Folder)에 안전하게 백업하고 동기화할 수 있도록 돕는 유틸리티 애플리케이션입니다. 본 앱은 어떠한 사용자 데이터도 외부 서버로 전송하지 않으며, 모든 백업본은 사용자의 개인 구글 계정 내에만 안전하게 보관됩니다.
